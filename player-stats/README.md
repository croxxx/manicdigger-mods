PlayerStats Mod by croxxx
=========================

Description
-----------
This mod lets you monitor user activities on your server.
Following command is added:

	/pstat [playername]

This displays following information:
 * first seen date and time
 * last seen date and time
 * time played on server (total ingame time)
 * number of block changes (built and deleted seperately listed)
 * last known IP adress


Installation
------------
1. Copy `PlayerStats.cs` into the folder `Mods\Fortress` of your Manic Digger installation directory.
2. Open `ServerClient.txt` in `UserData\Configuration` and add the privileges you need.
3. Start Manic Digger Server and enjoy user statistics.


Available privileges
--------------------
- pstat


Compatibility
-------------
- no conflicts with other mods
